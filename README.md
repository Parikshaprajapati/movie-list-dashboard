# Movie List Dashboard

A frontend-focused React application that displays and manages a list of movies with search and filtering capabilities.

This project is built to demonstrate strong fundamentals in:
- React (hooks, component structure)
- JavaScript (closures, immutability)
- State management
- Clean UI logic and data handling
